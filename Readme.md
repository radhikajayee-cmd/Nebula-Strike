## Overview

**Nebula Strike** is a lightweight front-end game that explores real-time interaction in the browser using vanilla JavaScript.
The project emphasizes clarity in structure, efficient DOM updates, and a visually consistent design system.

Rather than relying on frameworks, the goal here is to demonstrate how far **core web technologies** can be pushed to deliver an engaging gameplay experience.

## Core Features

* Continuous player movement with keyboard input
* Dynamic enemy spawning and interaction
* Collision detection and score tracking
* Smooth rendering using optimized DOM updates
* Responsive layout across screen sizes
* Minimal, neon-inspired visual system

---

## Tech Stack

* **HTML5** — semantic structure
* **CSS3** — layout, animations, visual system
* **JavaScript (ES6)** — game logic, state handling, interaction

---

## Architecture Notes

The project follows a simple, maintainable structure:

* Separation of concerns between structure, styling, and logic
* Event-driven input handling
* State updates reflected through controlled DOM manipulation
* No external dependencies

This approach keeps the project transparent and easy to extend.
